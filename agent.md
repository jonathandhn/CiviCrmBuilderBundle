# CiviCrmBuilderBundle

## Resume

Bundle Mautic qui ajoute des blocs et des outils GrapesJS relies a CiviCRM.

## Build

```bash
npm install
npm run build
```

## Points utiles

- Source frontend : `Assets/src/`
- Fichiers compiles : `Assets/dist/`
- Entree principale : `Assets/src/index.ts`
- Configuration Mautic : `Config/config.php`

## Convention

- Eviter les messages utilisateur avec emojis.
